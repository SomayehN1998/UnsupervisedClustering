# Unsupervised ML 

این مخزن شامل کدهای پایتون برای **۴ سوال پروژه یادگیری ماشین بدون نظارت (Unsupervised Learning)** است. هر سوال به طور مستقل در سلول‌های جداگانه پیاده‌سازی شده (مانند **Jupyter Notebook**)، اما همه در یک فایل اصلی به نام **ml_clustering.py** پشت سر هم آورده شده‌اند. پروژه‌ها عمدتاً حول **خوشه‌بندی** و استفاده از **الگوریتم‌های مختلف یادگیری بدون نظارت** می‌چرخند و برای دانشجویان و علاقه‌مندان به **یادگیری ماشین** مناسب هستند.

## توضیح پروژه‌ها 📋

### سوال ۱: **فشرده‌سازی تصویر با K-means**
- **توضیحات**: در این پروژه، از **الگوریتم K-means** برای خوشه‌بندی رنگ‌ها در یک تصویر استفاده می‌شود. این الگوریتم تصویر را به مجموعه‌ای از **خوشه‌های رنگی** تقسیم می‌کند، که در نهایت منجر به **فشرده‌سازی تصویر** می‌شود.
- **مراحل**:
  1. بارگذاری تصویر.
  2. اعمال **K-means clustering** بر روی رنگ‌ها.
  3. فشرده‌سازی و ذخیره تصویر جدید با رنگ‌های کمتر.
  4. مقایسه بصری و حجمی تصویر قبل و بعد از فشرده‌سازی.

### سوال ۲: **خوشه‌بندی DBSCAN روی داده‌های t-SNE**
- **توضیحات**: در این پروژه، از **الگوریتم DBSCAN** برای خوشه‌بندی داده‌ها استفاده می‌شود. داده‌ها ابتدا با استفاده از **t-SNE** (که ابزاری برای کاهش ابعاد است) به فضای دو بعدی منتقل می‌شوند و سپس خوشه‌ها شناسایی می‌شوند.
- **مراحل**:
  1. بارگذاری داده‌ها از یک **URL**.
  2. اعمال **DBSCAN** برای خوشه‌بندی داده‌ها.
  3. رسم **خوشه‌ها** و شناسایی **نویز** با رنگ‌های متفاوت.
  4. بررسی ویژگی‌های خوشه‌ها و تطابق آن‌ها با مدل DBSCAN.

### سوال ۳: **خوشه‌بندی سلسله‌مراتبی روی IRIS**
- **توضیحات**: در این پروژه از **خوشه‌بندی سلسله‌مراتبی** برای داده‌های **IRIS** استفاده می‌شود. این تکنیک به ما اجازه می‌دهد تا رابطه‌های سلسله‌مراتبی میان نمونه‌ها را مشاهده کنیم. همچنین از **Silhouette Score** برای انتخاب بهترین تعداد خوشه‌ها (K) استفاده می‌شود.
- **مراحل**:
  1. رسم **dendrogram** برای نمایش سلسله‌مراتب خوشه‌ها.
  2. محاسبه و نمایش **Silhouette Score** برای ارزیابی کیفیت خوشه‌بندی.
  3. استفاده از **scatter plot** برای نمایش داده‌ها و خوشه‌ها در فضای دو بعدی.

### سوال ۴: **مدل GMM روی IRIS**
- **توضیحات**: در این پروژه از **مدل Mixture of Gaussians (GMM)** برای خوشه‌بندی داده‌های **IRIS** استفاده می‌شود. این مدل با انتخاب تعداد مناسب خوشه‌ها، با استفاده از تکنیک‌های **elbow method** و **silhouette analysis** به خوشه‌بندی داده‌ها می‌پردازد.
- **مراحل**:
  1. انتخاب بهترین مقدار **K** با استفاده از **elbow method** و **Silhouette Score**.
  2. آموزش مدل **Gaussian Mixture Model (GMM)**.
  3. رسم **scatter plot** داده‌ها همراه با احتمالات تعلق به هر خوشه.
  4. رسم **Ellipses** برای نشان دادن نواحی مربوط به هر خوشه.

## هدف پروژه‌ها

هدف از این پروژه‌ها، آشنایی با **الگوریتم‌های خوشه‌بندی بدون نظارت** (Unsupervised Clustering Algorithms) مانند **K-means**, **DBSCAN**, **Hierarchical Clustering** و **GMM** است. علاوه بر این، این پروژه‌ها فرصتی عالی برای یادگیری مفاهیم مرتبط با **کاهش ابعاد** (مثل t-SNE) و **ارزیابی کیفیت خوشه‌ها** (Silhouette Score) به‌طور عملی فراهم می‌کنند.
